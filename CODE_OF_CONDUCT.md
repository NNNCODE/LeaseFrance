# Code of Conduct

## Our Commitment

In the interest of fostering an open, respectful, and constructive environment,
contributors, maintainers, and participants in this repository are expected to
make participation harassment-free for everyone.

## Expected Behavior

Examples of behavior that contribute to a positive environment include:

- being respectful in discussions and reviews
- focusing criticism on ideas, code, and evidence
- assuming good intent while still being technically precise
- accepting feedback gracefully
- keeping issues and pull requests scoped, actionable, and relevant

## Unacceptable Behavior

Examples of unacceptable behavior include:

- personal attacks, insults, or harassment
- discriminatory language or behavior
- trolling, baiting, or deliberately disruptive conduct
- publishing private information without permission
- repeated off-topic or bad-faith engagement

## Scope

This Code of Conduct applies within:

- GitHub issues
- pull requests
- review comments
- repository discussions and linked project spaces controlled by the maintainer

## Enforcement

The maintainer is responsible for clarifying and enforcing these standards.
Responses may include moderation, comment removal, issue locking, or refusing
future participation when behavior repeatedly harms the project environment.
