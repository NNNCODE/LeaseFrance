# Contributing

Thanks for your interest in Baillio.

This repository is currently public for portfolio and project showcase purposes.
It is not yet run as an open collaboration project with an active external PR
review queue.

## Current Contribution Policy

- Issues are welcome when they are specific, reproducible, and scoped.
- Pull requests should be opened only after prior discussion in an issue.
- Large unsolicited refactors are likely to be closed without review.
- Product direction and release sequencing remain maintainer-led.

## Before Opening An Issue

Please include:

- the goal or bug you observed
- exact reproduction steps
- screenshots or logs when relevant
- your environment details

For desktop/runtime issues, diagnostics and log excerpts are far more useful
than broad problem descriptions.

## Before Opening A Pull Request

Please keep changes narrow and explain:

- what problem is being solved
- why the current behavior is incorrect or incomplete
- how you validated the change

If your change touches packaging, auth, diagnostics, backup, licensing, or
document generation, include explicit test notes.

## Security Issues

Do not open public issues for security-sensitive reports.
Use the contact details in [SECURITY.md](SECURITY.md) instead.
